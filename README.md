# Cook County Weather Data Analysis (2016-2024)

## Proeject Overview
This project performs a **data analysis of weather patterns in Cook County, Illinois**, from **January 1, 2016, to December 31st, 2024**. The goal is to identify trends and evaluate whether certain weather events such as floods, winter storms, or changes in Lake Michigan surface water temperature have become more severe or frequent over time, or if they have remained relatively steady.

By analyzing a small sample of historical weather data, this project provides insights into patterns that may impact the region. The datasets are retrieved from the **NOAA Storm Events Database**.
